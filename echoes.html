Game description

2D platformer consisting of 2 simple levels

Platform: PC

Engine: Game Maker

Language: GML

Year: 2024

Project type: Team work (6 people)

A project done in a team of 6 people for an exam. The task was to make a small game (15-25 minutes of gameplay) in 8 weeks without using third-party assets.

The story of the game revolves around a young child who has lost all of their memory. The child is trying to find pieces of their memory by interacting with the local inhabitants â€“ frogs. The player can perform classic movements: moving forward, backward, and jumping. During the journey, the character can collect items, which can be exchanged with the frogs for pieces of memory. But there are also many dangers and enemies.

// Code example - inventory
    
// "Create" event

// size of the inventory
inventory_width = 1; 
inventory_height = 3;

// creating array
inventory = array_create(inventory_width * inventory_height, -1); // -1 is an empty cell

// the size of the cell
cell_size = 32;


///////////////////////////////////

// "Draw GUI" event



for (var row = 0; row < inventory_height; row++) {
    for (var col = 0; col < inventory_width; col++) {
        
        // get index from current cell
        var index = row * inventory_width + col;
        
        // get ID of the object
        var item_id = inventory[index];
        
        // get the coordinates of the object
        var cell_x = 10 + col * cell_size;
        var cell_y = 10 + row * cell_size; 
        
        // draw the object
        if (item_id != -1) { 
            draw_sprite(Collectible_item_sprite, item_id, cell_x + cell_size / 2, cell_y + cell_size / 2);
        }
		
		
		
    }
