Echoes of Fear

Game overview

Game description: 3D survival horror game

Focus: core gameplay systems, atmosphere, player survival mechanics

Platform: PC

Engine: Unreal Engine

Language: Blueprint scripting

Year: 2025

Project type: individual work, university project

Working on the project

This was my first project in Unreal Engine. My goal was to try a different engine since I had only worked with Unity before.

In the beginning of the game, player wakes up in a quiet cave, but shortly after, enemies begin approaching. The objective is to survive for 5 minutes. The player can fight using a sword, and if defeated, the game ends. Surviving the full duration results in a win.

What I did for the project

All assets were downloaded from free resources. The game was fully created using Blueprints, including:

– Enemy AI: attacking the player and spawning from both sides of the cave

– Player combat mechanics with a sword

– Countdown timer for the survival objective

– Health system for the player

– Flickering torch light to create atmosphere and tension

– Win and lose conditions
