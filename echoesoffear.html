Echoes of Fear

Game description

Game description: 3D survival game

Platform: PC

Engine: Unreal Engine

Language: Blueprint scripting

Year: 2025

Project type: Individual work

first project made in Unreal Engine. The player wakes up in a quiet cave, but after a few seconds they hear enemies approaching. The goal is to survive for 5 minutes. The player can fight using a sword, but if the enemies kill the player, the game ends. If the player manages to stay alive for the full 5 minutes, they win.

All assets were downloaded from free resources. The game was fully created using Blueprints, including:

-Enemy attack and player attack

-Timer

-Player health system

-Enemy spawning from both sides of the cave

-Flickering torch light

-Winning and losing conditions
